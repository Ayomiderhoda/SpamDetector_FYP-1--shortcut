About This System

This system classifies emails as Spam or Ham (legitimate) using a Naive Bayes Classifier trained on the UCI SMS Spam Collection dataset (5,574 messages). Features are extracted using TF-IDF Unigram and Bigram models, and similarity is measured using Dice Similarity (Dice, 1945) and Cosine Similarity (Salton et al., 1975).
